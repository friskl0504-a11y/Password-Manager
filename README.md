# Password Manager Download Site (Google Drive link)

เว็บนี้เป็น static site สำหรับให้คนกดปุ่มแล้วไปดาวน์โหลดไฟล์จาก Google Drive

## ใช้งานทันที (แบบโฟลเดอร์)
ตอนนี้ปุ่มจะเปิดลิงก์โฟลเดอร์:
https://drive.google.com/drive/folders/1LV5nqUct8Vwzu6W1XPrpfEXqBtIn9M5H

ผู้ใช้เลือกไฟล์ .exe แล้วดาวน์โหลดเอง

## แนะนำ: ทำให้ดาวน์โหลดตรง (Direct download)
1) ใช้ลิงก์แบบไฟล์: https://drive.google.com/file/d/FILE_ID/view
2) แปลงเป็น: https://drive.google.com/uc?export=download&id=FILE_ID
3) นำไปแทนที่ href ของปุ่มดาวน์โหลดใน index.html
